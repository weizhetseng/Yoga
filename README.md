
### Demo URL: https://yoga-ochre.vercel.app/
專案使用 Nuxt 3 框架建置，並搭配 Tailwind CSS 做為主要的前端設計樣式。

## 使用技術
- **Nuxt 3**：前端框架，用於建置伺服器端渲染的 Vue.js 應用。
- **Tailwind CSS**：實用的 CSS 框架，用於快速設計響應式網頁樣式。
